# AI 写作反模式索引

这个目录收集了 AI 写作中常见的"AI味"特征。每个文件包含：

- 特征描述
- 反面示例
- 改进写法
- 供 AI 参考的规则

## 反模式列表

| 文件 | 反模式 | 严重程度 |
|------|--------|----------|
| [emoji-abuse.md](emoji-abuse.md) | Emoji 滥用 | 高 |
| [cliche-phrases.md](cliche-phrases.md) | 套话陈词 | 高 |
| [template-structures.md](template-structures.md) | 模板化结构 | 高 |
| [transition-patterns.md](transition-patterns.md) | 机械过渡句 | 中 |
| [list-obsession.md](list-obsession.md) | 列表强迫症 | 中 |
| [fake-enthusiasm.md](fake-enthusiasm.md) | 虚假热情 | 中 |
| [over-explanation.md](over-explanation.md) | 过度解释 | 低 |

## 如何使用

1. **了解特征**：阅读每个反模式的描述，识别这些特征
2. **构建 Prompt**：将"规则"部分的内容加入你的 System Prompt
3. **对照检查**：写完后对照检查是否踩坑

## 贡献新反模式

发现新的 AI 写作特征？参考现有文件格式，提交 PR。
