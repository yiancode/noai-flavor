# 前端开发反模式索引

这个目录收集了 AI 在生成网站/前端代码时常见的"AI味"特征。

## 什么是前端的"AI味"？

就是那种一眼就能看出是 AI 生成的网站：
- 配色死板：蓝紫渐变色用到吐
- 布局死板：首屏放个大标题，下面三个卡片并排
- 字体死板：基本上就是 Inter、Roboto 等几种固定字体
- Emoji 泛滥：满屏幕都是表情图标
- 内容空洞：基本没有真实图片，文字风格也比较刻板

## 核心原因

AI 为什么会生成有"AI味"的网站？核心原因就是：**求稳**。

- AI 的训练数据中，Tailwind CSS、Shadcn UI 等主流库出现频率最高
- 这些库的默认配色就是蓝紫色渐变
- AI 学会了：用最常见的 = 最不容易出错
- 所以 AI 认为 "现代化网站 ≈ 蓝紫色渐变 + Hero 布局 + 三卡片"

## 如何使用

1. **了解反模式**：阅读每个反模式的描述，识别这些特征
2. **使用提示词**：将反模式规则加入到你的 Prompt 中
3. **主动约束**：明确告诉 AI 不要什么，而不只是说要什么

## 破局思路

从 "请求者" 变成 "指挥官"：

❌ 不要只说需求：给我做个网站

✅ 而是要明确要求：用深灰色背景、手绘图标、不对称布局、拒绝蓝紫色

用强有力的约束条件，逼着 AI 偏离它的舒适区。
