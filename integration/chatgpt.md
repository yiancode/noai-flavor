# ChatGPT 使用方式

## 方式一：自定义 GPT

创建一个自定义 GPT，在 Instructions 中添加 `prompts/base/anti-ai-core.md` 的内容。

### 设置步骤

1. 进入 ChatGPT → Explore GPTs → Create
2. 在 Instructions 框中粘贴：

```
你是一个写作助手。在生成内容时，请严格避免"AI味"，写出更自然、更有人味的文字。

### 禁止清单

**表达层面**
- 不用 emoji 做列表标记（禁止 ✅❌🎯📌💡）
- 不用"在当今社会"、"随着...的发展"开头
- 不用"综上所述"、"总而言之"结尾
...（完整内容见 prompts/base/anti-ai-core.md）
```

3. 保存并使用

## 方式二：对话前置

每次新对话开始时，先发送：

```
在这个对话中，请遵守以下写作规则：
[粘贴 anti-ai-core.md 的简化版]
```

## 方式三：作为提示词后缀

在每个写作请求后加上：

```
请用自然的语气写，避免 AI 味。不用 emoji 列表、不用套话、不用三段论。
```

## 效果验证

生成内容后，检查：

- [ ] 没有 ✅❌🎯 列表
- [ ] 没有"综上所述"
- [ ] 没有"首先其次最后"
- [ ] 开头不是"在当今..."
- [ ] 有句式变化

不达标就要求重写。
