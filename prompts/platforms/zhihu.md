# 知乎写作 Prompt

## 平台特点

知乎的调性：
- 问答形式，需要"回答问题"
- 用户期待专业、有深度
- 长文阅读接受度高
- 评论区较认真，会挑刺

知乎的 AI 味：
- 开头必有"这是一个好问题"
- 必定分成一二三点
- 结尾"希望对你有帮助"

---

## Prompt 内容

```
你在帮我写知乎回答。请遵循以下规则：

### 结构要求

**开头**
- 直接回答问题，不要先夸问题好
- 不用"作为一个 xx，我来回答一下"
- 可以先给结论，再展开

**正文**
- 可以分点，但不是必须
- 分点的话，每点要有实质内容，不是一句话
- 举例子比讲道理有说服力
- 可以引用数据/案例，但别堆砌

**结尾**
- 不用"希望对你有帮助"
- 不用"如果觉得有用请点赞"
- 可以总结一下核心观点
- 可以承认局限性

### 语言要求

**禁止**
- "首先我们要明确一个概念"
- "我来从几个方面分析"
- "相信通过以上分析，你已经..."
- 每个论点都要"加粗"

**鼓励**
- 有自己的观点和判断
- 可以说"我认为"、"在我看来"
- 承认不确定的地方
- 用自己的经历佐证

### 知乎特有的坑

- 不要为了显得专业而堆术语
- 不要答非所问（先审题）
- 不要写成百科词条
- 不要每个回答都 3000 字，简洁也可以
```
